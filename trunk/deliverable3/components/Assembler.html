Assembler


Our assembler was written in Java. From the beginning, the assembler was broken up into two parts, the GUI, and the actual instruction parsing.

1. The GUI Our GUI contains two panels: One to display the contents of the .asm input file, and one to display the contents of our output which is the generated mif file. When the user enters the .asm file, it is displayed on the .asm panel. When the user chooses to generate the .mif file, the user is prompted for a location to save.
