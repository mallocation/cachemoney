Sign Extend 32 Bit

The sign extender just takes a 16 bit number and extends it to a 32 bit number. The most significant bit is just appended to 16 times to the most significant bit side of the number.

VHDL Code

LIBRARY ieee;
USE ieee.std_logic_1164.all;

ENTITY SignExtend32bit IS
	PORT (		data_in		:	IN	STD_LOGIC_VECTOR(15 DOWNTO 0);
					data_out	:	OUT	STD_LOGIC_VECTOR(31 DOWNTO 0));
END SignExtend32bit;

ARCHITECTURE Behavior OF SignExtend32bit IS
BEGIN	
	data_out(15 downto 0) <= data_in;
	data_out(31 downto 16) <= (31 downto 16 => data_in(15));
END Behavior;
